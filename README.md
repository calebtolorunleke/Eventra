# Eventra Frontend Application

Eventra is a modern event management platform built with React and Vite. The application is designed to help users discover, create, and manage events through a clean, responsive, and intuitive user experience.

This project is being developed to demonstrate scalable frontend architecture, reusable component design, and modern React development practices.

## Features

- Event discovery
- Event details
- Event creation and management
- Responsive design
- Modern UI/UX
- Reusable React components
- API integration
- Authentication (planned)

- ## Tech Stack

- React    - Vite    - JavaScript (or TypeScript)
- Tailwind CSS    - React Router    - REST APIs    - Git & GitHub

- ## Getting Started

Clone the repository
```bash
git clone https://github.com/calebtolorunleke/Eventra.git

Install dependencies

npm install

Run the development server

npm run dev
```
